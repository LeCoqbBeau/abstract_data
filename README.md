# Abstract Data
The standard C++ containers have all a specific usage. 
To make sure you understand them, let’s re-implement them!

## Description
Abstract Data is a project in beta from the 42 school.\
The subject asks us to implement the sequence, associative and adaptors containers from the 1998 C++ standard.\
To help you to code all this, you can only use std::allocator, std::string, std::numeric_limits and all the std::iterator_tags.\
Thus, I have implemented the following containers: list, deque, vector, map, set, multimap, multiset, stack, queue, priority_queue, unordered_map, unordered_set, unordered_multimap, unordered_multiset.\
All the source code for these containers are located in the `include` directory.\
The second part of the requirements asks us to provide unit test for our containers to compare them to the STL.\
For these unit test I have used the [Google Test](https://github.com/google/googletest) library.
